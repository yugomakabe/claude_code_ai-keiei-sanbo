## MeetMemo — 会議を、実行力に変える

**バージョン**: v0.1 草案

**事業フェーズ**: シード期（MVP開発前）  

---

## 1. エグゼクティブサマリー

### 事業概要

**MeetMemo**は、会議の録音をAIが自動で議事録化し、タスクを抽出して担当者へ自動通知するSaaSである。議事録作成とタスク割り当てに毎回30〜60分を費やし、かつ口頭決定したタスクが実行されないまま次の会議を迎えるという、中小企業に蔓延する非効率を解決する。

### 解決する本質的課題

> 「会議で決まったことが、実行されない」

### ターゲット顧客

従業員50名以下の中小企業（国内約360万社）

### ビジネスモデル

月額3,000円 / ユーザーのサブスクリプション型SaaS

### 市場機会

国内AI会議ツール市場は音声認識普及とハイブリッドワーク定着を追い風に急拡大中。TAMは国内推定500億円規模。3年目ARR目標：**7,200万円**（600社×5ユーザー）

### 差別化の核心

既存競合（Notta・Otter.ai・LINE WORKS AiNote）はいずれも「議事録の作成・共有」で止まっており、**「タスクを担当者に自動通知し、実行を追跡する」機能を持つツールは現時点で市場に存在しない**。

## 2. 課題と解決策

### 2-1. 課題の深刻度

| 課題 | データ |
|---|---|
| 議事録作成に不満を持つ割合 | **60%以上**（SaaS比較サイト調査） |
| バーチャル会議で情報を保持できないリモートワーカー | **約60%**（Sonix 2026調査） |
| 生成AIを文書補助に使用中（日本） | **47.3%**（総務省 令和7年版情報通信白書） |
| 企業のAI活用率 | **78%**（業界調査） |

#### 課題の三層構造

```
表層（見えている問題）
  └── 議事録を作るのが面倒・時間がかかる

中層（本当の不満）
  └── 会議中に議事録を書くと議論に集中できない
      担当者によって議事録の質がバラバラ

深層（ビジネスへのダメージ）
  └── 口頭で決まったタスクが誰にも認知されず消滅
      「あの件どうなった？」確認がマネージャーを疲弊させる
      意思決定の実行率が低く、会議が無駄になる
```

### 2-2. 解決策

MeetMemoは「録音→議事録化→タスク抽出→担当者通知」を**ワンプロセスで自動化**する。

```
【会議中】
録音（スマホ/PC）または Web会議ツール（Zoom/Teams/Meet）に接続

        ↓ AI処理（Claude API + Whisper API）

【会議後・即時】
① 文字起こし（日本語精度98%超）
② 議事録の自動生成・整形
③ タスクの自動抽出（誰が・何を・いつまでに）
④ 担当者へSlack/Teams/メールで即時通知

        ↓ 継続追跡

【次回会議まで】
⑤ タスク進捗の可視化（完了/未完/遅延）
⑥ 次回会議前に進捗サマリーを自動生成
```

### 2-3. 提供価値の定量化

| 効果 | 試算根拠 |
|---|---|
| 議事録作成工数の削減 | 毎回30〜60分 → ほぼゼロ |
| 人件費換算（月4回会議・時給2,500円の場合） | **約2万円/人/月**の削減 |
| タスク漏れによる機会損失の低減 | 次回会議での「確認作業」（平均15分）の排除 |
| MeetMemoの月額コスト | **3,000円/ユーザー** |
| **ROI** | **投資対効果：約6.7倍** |

### 5-2. 各競合の詳細分析

#### Notta
- **概要**: 日本語精度98.86%を誇る業界トップクラスの文字起こしツール。58言語対応、東京データセンター保管。
- **料金**: 個人プレミアム 月¥1,185（年払い）〜
- **強み**: 日本語精度・コスパ・個人利用のシンプルさ
- **弱み**: タスク通知機能なし。チームでの実行管理に対応していない。個人向けの設計でチーム機能が薄い。

#### Otter.ai
- **概要**: 英語文字起こしに特化した米国発ツール。Zoom・Teams・Meetと連携。
- **料金**: Pro $8.33/月（年払い）、Business $20/月
- **強み**: 英語精度・リアルタイム文字起こし・西欧での知名度
- **弱み**: 日本語精度83%程度と低い。多言語対応・翻訳機能なし。日本語サポート体制が弱い。

#### LINE WORKS AiNote
- **概要**: LINEのビジネスチャットツール「LINE WORKS」が提供するAI議事録。旧CLOVA Note。
- **料金**: 個人ソロ 月¥1,440（年払い）、チーム 月¥19,800（年払い）
- **強み**: LINE WORKSユーザーへのシームレス統合・企業管理機能・セキュリティ
- **弱み**: 2026年4月にタスク抽出機能を追加したが**担当者への自動通知はなく「一覧表示のみ」**。LINE WORKSを使っていない企業には導入障壁。チームプランは月¥19,800と中小企業に割高。

### 5-3. MeetMemoの差別化軸

#### 差別化軸①：「通知まで一気通貫」という唯一の機能
競合はタスク抽出を追加し始めているが、**担当者へのPush通知（Slack/Teams/メール）まで自動化しているツールは2026年6月時点で存在しない**。この機能こそが「タスク消滅問題」を解決する核心であり、模倣コストも比較的高い。

#### 差別化軸②：中小企業に特化したUX・サポート・価格
- IT担当者不在でも3分で使えるセットアップ
- 日本語チャットサポート（営業時間内）
- AI導入補助金の申請サポートを提供（競合にない価値）
- 月¥3,000/ユーザーでROI約6.7倍の明快な費用対効果

#### 差別化軸③：「議事録→実行管理」のレイヤー拡張
議事録作成は競合と同水準で提供しつつ、**タスク進捗の追跡・次回会議前の自動サマリー生成**という「下流」機能を持つことで、「乗り換えコスト（スイッチングコスト）」を高め、長期的な解約防止（チャーン抑制）につなげる。
