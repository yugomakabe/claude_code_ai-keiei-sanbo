# AI 経営参謀システム

市場調査・分析・文章生成などを一貫して行う経営判断サポートシステム。

## プロジェクト概要

このシステムは以下の機能を一貫して提供する：

- **市場調査**: 競合・業界・トレンドの自動収集・整理
- **データ分析**: 収集データの定量・定性分析
- **文章生成**: レポート・提案書・サマリーの自動生成
- **経営判断サポート**: 分析結果に基づく意思決定支援

## ディレクトリ構成（予定）

```
ai-keiei-sanbo/
├── CLAUDE.md           # このファイル
├── src/
│   ├── research/       # 市場調査モジュール
│   ├── analysis/       # データ分析モジュール
│   ├── generation/     # 文章生成モジュール
│   └── pipeline/       # 一貫処理パイプライン
├── prompts/            # プロンプトテンプレート
├── outputs/            # 生成結果の出力先
└── tests/              # テストコード
```

## 技術スタック（予定）

- **言語**: Python / Node.js（未確定）
- **LLM**: Claude API (claude-sonnet-4-6 / claude-opus-4-8)
- **データ取得**: Web スクレイピング / 各種 API
